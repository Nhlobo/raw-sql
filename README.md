# Kutlwano Database-First Platform

Production-oriented pnpm monorepo for two independent React/Vite PWAs and one Fastify API, built around the existing PostgreSQL schema in `database/`.

## Commands

- `pnpm install`
- `pnpm validate`
- `pnpm typecheck`
- `pnpm build`

See `docs/ARCHITECTURE.md` and `docs/DEPLOYMENT.md`.
